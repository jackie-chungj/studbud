# StudBud

StudBud is a design-led productivity dashboard built to support focused study sessions through a single, calm interface.

It combines a Pomodoro timer, stopwatch, Covey matrix task planning, embedded music, and a lightweight dictionary into one responsive web app. The goal was to create a study tool that feels structured and useful without becoming visually overwhelming.

## Features

- Pomodoro timer with work/break states
- Animated session progress line
- Optional sound feedback
- Stopwatch with lap tracking
- Covey matrix task board
- Embedded Spotify study music
- Dictionary lookup
- Responsive layout for desktop and mobile

## Project context

StudBud evolved from my original frontend build and UI direction. I used AI-assisted development workflows to help debug, refine interactions, improve responsiveness, and bring the product closer to the experience I wanted.

Rather than treating AI as a replacement for design or implementation, I used it as a development partner to speed up iteration while keeping product direction, UX decisions, and visual refinement intentional.
